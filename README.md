# Java Sorting Strategy App

Учебный командный проект для реализации алгоритмов сортировки кастомных объектов с использованием паттернов проектирования.

## 🛠 Технологии
- Java 17
- Gradle

## 📦 Функционал
- Сортировка кастомных объектов
- Паттерн Strategy (разные алгоритмы сортировки)
- Паттерн Builder (создание объектов)
- Валидация данных
- Варианты ввода:
    - вручную
    - случайная генерация
    - загрузка из файла

## 🧠 Архитектура
Проект построен с разделением ответственности:
- модели (models)
- стратегии сортировки (strategy)
- сервисы (service)
- ввод/вывод (io)

## 🚀 Запуск
```bash
./gradlew run
```
